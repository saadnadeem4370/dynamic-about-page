# Dynamic About Page with Team Management System

## 📌 Project Description
This is a full-stack web application that displays a dynamic About page with team management features. It includes an admin panel to add, edit, and delete team members.

## 🚀 Features
- About Company Section (Mission, Vision, Services)
- Dynamic Team Members (from JSON database)
- Admin Panel (CRUD Operations)
- Search Team Members by Name
- Filter by Role (Developer, Designer, Manager)
- Dark/Light Mode
- Responsive Design

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: JSON File
